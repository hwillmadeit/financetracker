# 개인 재정 관리

보험, 구독, 교육, 대출, 투자, 생활비, 경조사비를 한눈에 관리하는 웹앱입니다.

## Vercel 배포 방법

### 방법 1 — Vercel CLI (권장)
```bash
npm install
npx vercel
```

### 방법 2 — GitHub 연동
1. 이 폴더를 GitHub 저장소에 push
2. [vercel.com](https://vercel.com) → New Project → GitHub 저장소 선택
3. Framework: Next.js (자동 감지) → Deploy

### 로컬 실행
```bash
npm install
npm run dev
# http://localhost:3000
```

## 기능
- **투자** — 종목별 매수/매도 거래 기록, 평가액 갱신, 수익률 자동 계산, 이율/만기 표시 (예금/채권)
- **보험** — 월 보험료, 만기일, 갱신일 관리
- **구독** — 서비스별 월 구독료, 다음 결제일 관리
- **교육** — 과정별 월 비용, 수강 기간 관리
- **대출** — 잔여 원금, 금리, 월 상환액, 만기일 관리
- **생활비** — 고정 지출 예산 + 경조사비 건별 기록 (연간 예산 연동)
- **요약** — 월 지출 추이 차트, 투자 자산 배분 차트
- **검색** — 전체 항목 이름/메모/태그 통합 검색
- **CSV 내보내기** — 각 탭별 엑셀 호환 CSV 다운로드

## 데이터 저장
모든 데이터는 브라우저 `localStorage`에 저장됩니다. 서버로 전송되지 않습니다.
